Inline: Out of Time

Fall 2022 - Spring 2023

Overview

Inline: Out of Time is an adrenaline-fueled 2D platformer packed with high-speed deliveries, urban parkour, and rail-grinding.


Role: Tools Programmer

Engine: Custom C++ Engine

Team Size: 11 (5 programmers, 3 artists, 3 designers)

Steam Link

Contributions

Implemented and expanded the Spine API library to allow artists to add fully animated assets to the engine without touching engine code.

Added additional Spine API library features near the end of the development cycle to allow for the randomization and animation of NPC characters.

Implemented core engine components with a focus on art related features such as Transform, Sprite, and Animation Manager classes.

Acted as liaison between art and programming teams to balance expectations and ensure task completion.

Participated in pairs programming with our gameplay programmer to implement player animations and player controller using a state-machine.

Collaborated on the game development cycle through weekly sprints, standups and code reviews.

Received 50+ positive reviews within 1 year of the game being released on steam.
